    ** Linux(Mac) uses the Terminal software to access
    the command line interface. **

cd

Change folder

cd /

Change to the root drive (default drive)
(note the space after "cd")

cd ~

Change to the current folder
(note the space after "cd")

cd ..

Change to the parent folder
(note the space after "cd")

clear

Clear the screen

cp

Used to copy files from disk to disk or create a second copy of file on a single disk.

mv

Used to move files from disk to disk or rename a file on a single disk.

rm

Delete a file

rm -r

Delete a directory and all files & subdirectories

rmdir

Delete an empty directory

ls

List/view/display the contents of a directory

ls -a

List/view/display all hidden files in a directory

ls -d */

List/view/display all directories

mount

Eject driver from your computer.

mkdir

Create a folder/subfolder

rd

Delete a folder

pwd

Display the current Directory

diskutil list

Show all drives (and information about the drive) that are connected

*

Wildcard symbol
