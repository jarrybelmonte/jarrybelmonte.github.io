Windows CMD Commands





Home Linux Windows Scenario

    ** Windows uses the CMD software to access the
    command line interface. **

cd

Change folder

cd\

Change to the root drive (default drive)

cd.

Change to the current folder

cd..

Change to the parent folder

cls

Clear the screen

copy

Used to copy files from disk to disk or create a second copy of file on a single disk.

move

Used to move files from disk to disk or rename a file on a single disk.

del

Delete a file

dir

Show the contents of the current folder

dir /ad

Show only subfolders in the current folder

dir /p

Show files in current folder and pause between screens

dir /w

Shows files in current folder listed in wide format

erase

Delete a folder

format

Format a drive

md

Create a folder/subfolder

move

pwd

rd

Delete a folder

ren

Rename a file or folder

driveletter :

Change to another drive

*

Wildcard symbol
